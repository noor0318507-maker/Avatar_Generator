# Advanced Avatar Generator 🎲✨

A sleek, modern web application that generates unique, customizable avatars using the **Dicebear API**. Users can explore various designer art styles, enter random seeds, choose specific export formats, track history, and save bookmarks locally.

---

## 🚀 Features

* **Diverse Art Styles:** Switch dynamically between 17 unique art styles including Robots, Anime, Pixels, Humans, and Doodles.
* **Seed-Based Generation:** Type any phrase, word, or name to generate a highly unique configuration.
* **True Randomizer:** Spin the dice to mix and match randomized vector patterns instantaneously.
* **Flexible Format Downloads:** Export your custom avatars directly to high-quality **SVG** vectors or standard **PNG** formats.
* **Resolution Slider:** Scale your downloads dynamically anywhere from **128px up to 1024px**.
* **Local History Tracking:** Keep records of your last 6 generated setups so you can restore recent favorites with one click.
* **Persistent Bookmarks System:** Save preferred iterations instantly via `localStorage` integration.

---

## 🛠️ Built With

* **HTML5** – Structured semantic canvas elements.
* **CSS3** – Custom properties, Flexbox alignment, CSS Grid positioning, and smooth modern micro-animations.
* **JavaScript (ES6+)** – Asynchronous networking, custom event handling, Base64 conversion pipelines, and native Canvas API handling.
* **Dicebear API (v7.x)** – Open-source vector design generation engine.
